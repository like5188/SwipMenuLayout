# SwipMenuLayout

SwipMenuLayout支持任何View的侧滑

## 使用方法：

1、引用

在Project的gradle中加入：
```groovy
    allprojects {
        repositories {
            ...
            maven { url 'https://jitpack.io' }
        }
    }
```
在Module的gradle中加入：
```groovy
    dependencies {
        compile 'com.github.like5188:SwipMenuLayout:1.0.0'
    }
```
